# Metric definitions

| Metric | Definition | Evidence caveat |
|---|---|---|
| Known spending/sales | Sum of strictly positive reported fees | Not total market activity |
| Average/median transfer age | Completed age on transfer date | Missing DOB excluded |
| Domestic share | Transfers whose mapped source/destination club country is Portugal | Unknown club countries remain unknown |
| U21/U23 players used | Distinct players below threshold on match date with appearances | Not academy graduates |
| U21/U23 minute share | Youth minutes / all recorded club minutes | PO1 source coverage only |
| Minutes-weighted age | Sum(age × minutes) / total minutes | Historical event age |
| Development Index | Configurable youth exposure composite | Directional, not universal |
| Known-fee coverage | Records with a positive reported fee | Shows denominator quality |
| Minute concentration | Top-five minutes / all club-season minutes | Screening indicator |
| Position depth | Players reaching 450 minutes in broad position | Broad source position |
